Project: Restaurant delivary fee lookup
Description: Reads a .csv file that includes address and fee data. Provides command line lookup functionality
	     for addresses and their delivary fee in the file.
Invocation: delivaryfee <.csv file>.

In-depth specs:
	- Supports modification to the .csv file in real time.
	- Automatic closing of file to prevent data loss
	- Extremely fast address lookup 
	- Low memory usage
	- Simple and concise interface


Undetermined:
	- Advanced function switches
	- Automatic backing up of file before writing
	- Dynamic address fields (other fields like "notes" and "name")
	- Clearing console screen upon command
	- Printing usage log file on exit
